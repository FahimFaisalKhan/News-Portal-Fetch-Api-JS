Q1. What is the difference between 'var' , 'let' and 'const'?

Ans: Difference between var,let and const:

Variables declared with var and const are scoped to the immediate function body. Variables declared with the var keyword are hoisted. Hoisting means that the variable can be accessed in their enclosing scope even before they are declared. Variables declared with the let keyword are block-scoped, which means the variables will have scope to the immediate enclosing block.

Q3. What is the difference between map, forEach , filter , find?

Ans: Difference between map, forEach ,filter ,find:

Map forEach,filter,find all of these 4 are array methods . map executes some function on an array by a for loop and returns another another nodified array. forEach does the same thing as map but the difference is it does not return anything , it justs executes the code in the function without any return value. Filter is also an array method which goes through the array with a function by for loop , filter returns all values that satisfies specified condition/conditions. Find does the same thing as filter but it only returns the first value that meets specified condition/conditions.
